# AI提示词收藏网站

这是一个简单的AI提示词收藏网站，灵感来源于 [AI提示词收藏](https://anaitsc.xiaofeipandian.cn/)。

## 功能特点

- 📝 精选AI提示词展示
- 🏷️ 提示词分类浏览
- ⭐ 收藏喜欢的提示词
- 🎨 简洁美观的界面设计
- 📱 响应式布局，支持移动端

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
- localStorage (本地存储收藏数据)

## 项目结构

```
/
├── index.html          # 主页面
├── css/               # 样式文件
├── js/                # JavaScript文件
├── data/              # 数据文件
└── assets/            # 图标等资源
```
